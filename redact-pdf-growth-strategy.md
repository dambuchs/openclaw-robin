### Redact PDF AI Differentiators (vs. all of the above)

1. **AI-powered PII detection** — not just manual blackout boxes
2. **EU/Swiss data residency** — massive for GDPR-regulated markets (Switzerland, France, Germany, Belgium)
3. **Page-based pricing** — more transparent and fairer for docs with many pages
4. **Excluded + Always-Redact Terms** — no competitor has this level of control
5. **100+ languages** — unique for multilingual European markets
6. **REST API** — only enterprise tools offer this; Redact PDF AI will offer it at SMB pricing
7. **Simplicity** — clients love the UX vs. complex legacy tools

### Key Positioning Statement (to use everywhere)
> _"The only AI PDF redaction tool built for European compliance — GDPR, HIPAA-ready, Swiss-hosted, with automatic PII detection that actually understands your documents."_

## 4. AI Search / GEO Strategy (ChatGPT, Claude, Perplexity)

### Why this matters
When someone asks ChatGPT "what's the best tool to redact PDFs in Europe", you want Redact PDF AI to come up. Right now, it probably doesn't. Redactable does — it has G2 reviews, backlinks, and is mentioned across dozens of "best redaction tool" articles.

### How LLMs pick tools to recommend
1. **High-quality long-form content** that answers the exact question
2. **Mentions in third-party articles** ("best X tools" roundups)
3. **G2/Capterra reviews** — LLMs scrape these
4. **Reddit threads** — perplexity.ai pulls from Reddit heavily
5. **Structured data** (schema.org markup, llms.txt ✅ already done)

### Actions to take

**Short-term (1-2 months):**
- Get listed on G2 + Capterra + AlternativeTo → LLMs cite these
- Create Reddit presence: answer questions in r/legaladvice, r/gdpr, r/notary about document privacy + mention the tool naturally
- Reach out to bloggers who wrote "best PDF redaction tools" articles and ask to be included

**Medium-term (3-6 months):**
- Publish 4-6 long-form blog posts targeting questions LLMs answer:
  - "Best PDF redaction tools for GDPR compliance (2025)"
  - "How to redact a PDF — complete guide"
  - "GDPR and document sharing: what you need to redact"
  - "PDF redaction for notaries: step-by-step guide"
- Get featured in 2-3 European tech/legal publications
- Build up G2 reviews to 10+

**llms.txt is already good** — make sure it's kept up to date with new features and differentiators.
